\# VoiceAuthentic - Phase 1 Component Documentation



\## Component 1: Dataset Organization



\### Purpose



This component organizes the audio dataset into separate categories for real and fake voices.



\### Input



Audio files (.wav)



\### Structure



dataset/

├── real/

└── fake/



\### Output



Structured dataset ready for analysis and machine learning.



\### Importance



Proper organization ensures easy access, processing, and labeling of audio samples.



\---



\## Component 2: Dataset Inspection Module



\### File



phase1/dataset\_inspection.py



\### Purpose



To inspect the dataset and gather basic information about audio files.



\### Functions Performed



\* Counts audio files

\* Reads sample rate

\* Calculates duration

\* Displays dataset statistics



\### Input



WAV audio files



\### Output



\* Number of files

\* Sample rate

\* Duration

\* Audio metadata



\### Libraries Used



\* os

\* scipy.io.wavfile



\### Importance



Helps verify dataset integrity before feature extraction.



\---



\## Component 3: Waveform Analysis Module



\### File



phase1/waveform.py



\### Purpose



To visualize audio signals in the time domain.



\### Process



Audio File

→ Read Samples

→ Plot Waveform

→ Save PNG



\### Input



WAV audio file



\### Output



\* real\_waveform.png

\* fake\_waveform.png



\### Libraries Used



\* scipy

\* matplotlib



\### Importance



Provides visual representation of speech amplitude changes over time.



\---



\## Component 4: Spectrogram Analysis Module



\### File



phase1/spectrogram.py



\### Purpose



To visualize frequency distribution of audio signals over time.



\### Process



Audio File

→ Frequency Analysis

→ Spectrogram Generation

→ Save PNG



\### Input



WAV audio file



\### Output



\* real\_spectrogram.png

\* fake\_spectrogram.png



\### Libraries Used



\* scipy

\* matplotlib



\### Importance



Spectrograms reveal hidden frequency characteristics and are commonly used for deepfake detection.



\---



\## Component 5: MFCC Feature Extraction Module



\### File



phase1/mfcc.py



\### Purpose



To extract Mel-Frequency Cepstral Coefficients (MFCCs) from audio signals.



\### Process



Audio File

→ MFCC Extraction

→ Feature Visualization

→ Save PNG



\### Input



WAV audio file



\### Output



\* real\_mfcc.png

\* fake\_mfcc.png



\### Libraries Used



\* librosa

\* matplotlib



\### Importance



MFCCs represent speech characteristics and serve as input features for machine learning models.
